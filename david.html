MySome:

The mysome method is a recreation of the JavaScript method Array.prototype.some(). It checks whether there is at least 1 element in the array which passes the test given by the function. If there is at least 1 element for which the function returns true then it returns true. Otherwise it returns false. This does not modify the array in any way.

Input format: comma separated items like 1,2,3,4,5,6

MyReduce:

The myReduce method is a recreation of the JavaScript method Array.prototype.reduce(). It performs the function on each element in the array passing in the return value into the function for the next element in the array. The final output is the result of these function calls on every element in the array (after the last element of the array is being performed on by the function, the return value is the final result).
